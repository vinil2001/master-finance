Для каждой сущности, с которой нужно работать из MySql-базы должен быть набор CRUD-запросов :
GetAll - получить список всех записей
Get(int id) - получить запись по Id
Create(item) - добавить запись
Edit(item) - изменить запись, передавая все имеющиеся поля
Delete(int id) - удаление (обсудить необходимость)

Алгоритм работы аутентификации. 
Приложение отправляет в Web.api запрос в некий метод авторизации, передавая логин и пароль пользователя. 
Web.api проверяет логин и пароль и генерирует JWT-токен, возвращая его в ответ приложения. 
Далее, приложение отправляя любые другие запросы вместе с параметрами этих запросов отправляет полученный
ранее JWT-токен. 
Если JWT-токен корректный, Web.api выполняет действия соответствующие полученныму запросу.
